# Cron Expression Parser

A command line tool for parsing UNIX cron expressions.

Expected Input: `cron-expression some_command`

Example: `5 * * * MON /usr/bin/find`

## How to run the project

## Install Dependencies

```sh
npm install
```
## Run the project

```sh
npm start
```

| Command | Description |
